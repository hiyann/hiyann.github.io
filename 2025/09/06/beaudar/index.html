安装配置beaudar评论插件

一款基于issues的免费评论插件

1. 设置仓库

项目仓库地址：https://github.com/beaudar
选择 Beaudar 将要连接的Github仓库。
确保仓库是公开的，否则您的读者将无法查看 Issue(评论)。
确保 Beaudar app 已在仓库中安装，否则用户将无法发表评论。
打开网站 https://github.com/apps/beaudar 安装。

在仓库中新建一个beaudar.json文件，内容是你的网站地址。

如果你的仓库是一个分叉，请到设置中，确保 Issues 功能已打开。

仓库:
例：beaudar/beaudar
一个 public 的 GitHub 仓库。这是将发布博客文章 Issue（评论）的地方。

分支 (可选):
默认：master/main
仓库的分支名，用于校验仓库 beaudar.json 配置，以保证评论不会随意添加到目标仓库中。

2. 配置代码

官网 https://beaudar.lipk.org/ 配置代码

最后用Github账户密码登录即可评论。
